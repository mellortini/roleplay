================================================================================
                            NASTĘPNE KROKI
================================================================================

1. Przeczytaj START_HERE.md (5 minut)

2. Wygeneruj JWT_SECRET:
   node generate-jwt-secret.js

3. Ustaw zmienne na Railway:
   - SERVER: DATABASE_URL, JWT_SECRET, CLIENT_URL, NODE_ENV
   - CLIENT: VITE_API_URL, VITE_SOCKET_URL

4. Zrób redeploy:
   git add .
   git commit -m "Fix deployment"
   git push

5. Testuj aplikację:
   - Backend: https://roleplayserver.up.railway.app/api/health
   - Frontend: https://roleplayclient.up.railway.app

6. Jeśli coś nie działa:
   - Sprawdź logi: Railway → Deployments → Logs
   - Sprawdź FAQ_DEPLOYMENT.md
   - Sprawdź DevTools (F12)
